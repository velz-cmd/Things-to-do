## Full definition

RESOLVE is the **intelligence and execution layer for the open internet**. It continuously understands how value flows across open communities, explains what matters, recommends decisions, and executes capital movement when humans approve.

The open internet is the largest economy in the world, but nobody can see it as one system. RESOLVE is the intelligence layer that understands it.

## The capability order

Settlement is the **last** capability — not the first.

```
Observe  →  Understand  →  Recommend  →  Execute
```

| Stage | User experience |
|-------|-----------------|
| **Observe** | Sensors ingest value across ecosystems (invisible) |
| **Understand** | Mission answers any question about open communities |
| **Recommend** | Evidence-backed proposals — allocations, risks, gaps |
| **Execute** | Prepare → Approve → Settle (only when user wants action) |

Intelligence is the product. Execution is optional.
