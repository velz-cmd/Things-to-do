# Mission Tab — Ship Spec

**North star:** Mission turns proof into a named, simulatable, approvable money plan — in one place.

## The wow loop (60 seconds)

```
Pay signal (optional) → Blueprint payees → Simulate → Authorize → Mission receipt
```

## vs Arc hackathon demos

| Demo | Stops at | Mission continues |
|------|----------|-------------------|
| Obolus | Source toll + citation receipt | Multi-payee Blueprint + batch settle |
| Inktoll | Article unlock + publisher feed | Intel → settlement package |
| Gaffer | Branch steer + moment receipt | Paid context → capital design + memory |

## Tab ownership

- **Discover** finds gaps · **Mission** decides · **Capital** moves money
- Mission never hosts opportunity board or primary wallet UI

## Demo paths

1. `/mission` → **Fund value** tile → Blueprint (no chat essay)
2. `/mission?scope=react` → scoped fund prompt → Blueprint
3. Agent intel → pay → Blueprint auto-fills → simulate → authorize
4. `/mission/report/[id]` → decision receipt
